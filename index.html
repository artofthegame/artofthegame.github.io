The Art of the Game

UC Berkeley Spring 2018 Decal

This Art of the Game is a student run DeCal built for anyone with an interest and love for strategy games. This course was designed for people with the ambition to build game skills and think analytically about the strategy behind each game. While this course has a high level of gameplay, we welcome all new students as well to learn the rules and basics of the game. This semester we will cover a range of standard deck card games as well as board games.
