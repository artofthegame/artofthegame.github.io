Welcome to The Art of the Game

The Art of the Game is a student run DeCal built for anyone with an interest and love for strategy games. This course was designed for people with the ambition to build game skills and to think analytically about the strategy behind each game. While this course has a rather rigorous level of gameplay, we welcome everyone to learn the rules and basics of the game. This semester we will cover three types of games: standard deck card games, board games, and social deduction games.
